
Kibs Robots
=======

Neoforge tech mod that adds robot npcs that do tasks for you. Farmers will automatically harvest and plant crops and lumberjacks will chop and plant trees for you. They will automatically charge at stations you set for them aswell as depositing their loot. Very basic energy generators and works with other tech mods :)

How to Use
=======

To set a robots working area you need to craft a robot wand, use it to set the area, each robot has a different max area. Then you can shift right click on a station to set where the robot would charge up and give loot.

Features
=======
    Adds copper through netherite robots
    Lumberjack robots chop and plant trees
    Farmer robots harvest and plant crops aswell as including pumpkins and melons
    Solar panels that get energy and output through the bottom which works with other tech mods
    Robot station that allows robots to charge up and deposit loot automatically
    Everything works with other tech mods and I reccomend using one for more power generation options!
    Adds cobalt ore for some recipes
